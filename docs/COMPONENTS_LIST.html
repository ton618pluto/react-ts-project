⚓自定义 Hooks

封装业务逻辑的 React Hooks，用于数据加载、状态获取、快捷键绑定等
Hook 名称 模块归属 功能说明
useGetUserInfo Hook 从 Redux store 中获取当前用户信息（username、nickname）
useLoadUserData Hook 页面初始化时自动加载用户数据，如果未登录则请求后端获取，登录后将用户信息存入 Redux
useNavPage Hook 路由守卫 Hook，根据用户登录状态自动跳转：已登录访问登录页→跳转到管理列表；未登录访问需要登录的页面→跳转到登录页
useLoadQuestionData Hook 问卷编辑器页面加载时，根据 URL 中的问卷 ID 请求后端 API 获取问卷数据，然后重置 Redux 中的 components（组件列表）和 pageInfo（页面信息），并清空撤销历史
useLoadQuestionListData Hook 问卷管理列表页核心 Hook，监听 URL 搜索参数变化（keyword、page、pageSize），自动请求后端获取问卷列表数据，支持分页和搜索
useGetComponentInfo Hook 获取当前问卷编辑器中的组件信息，包括：componentsList（组件列表）、selectedId（选中ID）、selectedComponent（选中组件）、visibleComponnents（可见组件）
useGetPageInfo Hook 获取问卷页面基本信息（标题、描述、JS、CSS、是否发布等），用于编辑器页面设置和发布
useBindCanvasKeyPress Hook 编辑器画布快捷键绑定，使用 ahooks 的 useKeyPress 实现：
  • Delete/Backspace：删除选中组件
  • Ctrl+C：复制选中组件
  • Ctrl+V：粘贴组件
  • ↑/↓：选中上一个/下一个组件
  • Ctrl+Z：撤销
  • Ctrl+Y / Ctrl+Shift+Z：重做
